# docker-zsh-lab
My using test docker image.

## Build

```shell
$ docker build -t <yourname>/docker-zsh-lab .
```

## Run

```shell
$ docker run -it --rm <yourname>/zsh-lab
```

or

```shell
$ docker run -it --rm nnao45/zsh-lab:latest
```
